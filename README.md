#### Install

```sh
 npm install -g chaser-cli
```

#### Usage

use **init** commander to init a project

```bash
# init a project with template
chaser init [projectName]

# example:
# init SaaS project
chaser init SaaS
# init at current directory
chaser init .
```

and then choose a template you like
